Öncelikle projem server ve client olarak ikiye ayrılıyor. İkisi de ayrı ayrı terminaller kullanılarak çalıştırılmalı.
	Server kısmı node.js,
	Client kısmı React ile oluşturulmuştur.
	
Server kısmını çalıştırma:
	- Çalıştırılma ortamında node.js yüklü olmalı.
	- Projenin ./server klasöründe terminale "npm start" komutunu girerek çalıştırılmalı.
	
Client kısmını çalıştırma:
	- Projenin ./client klasöründe terminale "npm run dev" komutunu girerek çalıştırılmalı.
	- Proje çalıştırılırken oluşturulan url portunun 3006 olduğundan emin olunmalı. (Terminalde görebilirsiniz)
